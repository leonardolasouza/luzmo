# AI Roadmap reference

This is the **primary specification file** for this project.

Any AI assistant or IDE must read this file first and follow the reading order below.

As there are several .md files, this one gives a roadmap and sequence in how to read each file.

On the root folder of this project, there are 2 folders:
- backend: It will have the backend for this application.
- frontend: It will have the frontend for this application.
